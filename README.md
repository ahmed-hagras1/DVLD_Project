# 🚘 Driving License Management System (DVLD)

This is a C#-based application designed to automate and manage the services of a government driving and vehicle license department.

---

## 🔐 Authentication

- Secure login using **username and password**
- Option to **save login credentials**
- Ability to **log out** at any time

---

## 👤 People Management

- View list of registered people
- Add, edit, or delete person records
- Ensure national ID uniqueness

---

## 👥 User Management

- Add, edit, or delete system users
- Link users to real people in the system

---

## 🧑‍✈️ Driver Management

- Manage registered drivers
- Add new drivers after license issuance
- Edit or delete driver profiles

---

## 📄 License Requests

- Submit requests for:
  - Local licenses (7 types)
  - International licenses
- Prevent duplicate/incomplete requests
- Link each request to a person

---

## 📝 License Testing

- Schedule and manage:
  - Vision Test
  - Theory Test
  - Practical Driving Test
- Track test results and allow re-tests

---

## 🪪 License Issuance

- Issue license after all tests are passed
- Store license info: class, dates, holder, etc.
- Prevent duplicate licenses of the same class

---

## 🚫🔓 License Detain / Release

- Detain license with reason and fine
- Release license after penalty is paid
- Record and track suspension history

---

## ♻️ License Renewal

- Renew existing licenses
- Require vision test before approval

---

## 📦 Technologies Used

- **C# / .NET**
- **SQL Server**
- **Windows Forms (or WPF)**
- **ADO.NET**
- **GitHub for version control**
